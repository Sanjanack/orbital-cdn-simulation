#### **1. `main.py`** (Basic Prototype)
- **Purpose**: Initial basic SimPy simulation
- **Features**:

  - Random content selection from catalog
- **Usage**: `python main.py`
- **Output**: Console messages showing user requests

#### **2. `content_data.py`** (Content Catalog)
- **Purpose**: Defines the content catalog for simulation
- **Features**:

  - Each item has ID and size
- **Usage**: Imported by other simulation files
- **Content Types**: Video, Image, Document, Audio, Game

#### **3. `satellite_cdn_simulation.py`** (Core LRU Simulation)
- **Purpose**: Main satellite caching simulation with LRU algorithm
- **Features**:
  - `Content` dataclass with popularity
  - `Satellite` class with LRU cache using `OrderedDict`
  - User request process with popularity-based selection
  - CSV logging of all requests
  - Cache hit rate calculation
- **Usage**: `python satellite_cdn_simulation.py`
- **Output**: `simulation_log.csv` + console summary

#### **4. `enhanced_satellite_cdn.py`** (Advanced Simulation)
- **Purpose**: Enhanced version with multiple users and detailed metrics
- **Features**:
  - Multiple concurrent users
  - Performance monitoring
  - Detailed content types
  - Two CSV outputs (requests + performance)
  - Real-time metrics tracking
- **Usage**: `python enhanced_satellite_cdn.py`
- **Output**: `satellite_cdn_requests_*.csv` + `satellite_cdn_performance_*.csv`
